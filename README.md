# COVID-19-Global-Dashboard-PowerBI


## Project Overview

This project presents an interactive Power BI dashboard analyzing the global impact of COVID-19 using the Johns Hopkins University CSSE dataset.

The dashboard provides key performance indicators, trend analysis, country comparisons, and geographical visualizations to support data-driven decision-making.

---

## Objectives

- Analyze global COVID-19 data
- Track infection trends
- Compare countries
- Visualize geographical spread
- Build an interactive dashboard

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

---

## Key KPIs

- Total Confirmed Cases
- Total Deaths
- Active Cases
- Recovery Rate
- Countries Affected

---

## Dashboard Features

- Interactive Filters
- KPI Cards
- Line Charts
- Bar Charts
- Bubble Map
- Donut Chart
- Dynamic Slicers

---

## Key Insights

- COVID-19 affected over 200 countries.
- Global confirmed cases peaked in 2022.
- The United Kingdom recorded the highest cumulative confirmed cases in this dataset.
- Interactive filtering allows users to explore trends by country and reporting period.

---

### Recommendations
Based on the dashboard insights:

* Governments should continue strengthening disease surveillance systems.
* High-risk countries require sustained investment in healthcare infrastructure.
* Real-time dashboards should be adopted for faster public health decision-making.
* Public awareness campaigns remain essential to improve preparedness for future outbreaks.
* Interactive dashboards enable quicker.
